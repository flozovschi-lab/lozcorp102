# LOZ CORP RELOCATIONS — White Minimal Website (Static)

This is a multi-page static website (HTML/CSS) with a clean white theme and calm minimal illustrations (SVG).
Deploy on Vercel by importing the GitHub repo.

## Deploy
1) Upload all files to the root of a GitHub repo.
2) Vercel → New Project → Import repo → Deploy.
3) Add your domain in Vercel → Settings → Domains.
